Live Translator for Discord

Real-time translation built natively for Discord voice calls.

livecalltranslator.netlify.app

Open Live Website Contact

Problem & Solution

Cross-language voice chats lose momentum when translation lags or lives outside Discord. Live Translator provides real-time translation with native Discord integration, ensuring no context switching and clear in-channel visibility.

Technical Approach

Streaming STT (Deepgram) feeds a translation worker; inline bot posts interim + final lines. Optional captions overlay mirrors chat feed for on-screen visibility.

Key Constraints

  • <250ms interim response, sub-second final translations
  • Works inside Discord chat; keeps moderator controls intact
  • Reconnection guards + jitter buffers smooth dropouts
  • Rate-limit backoff and interim/final dedupe reduce chat spam

Privacy & User Experience

Per-user language preferences ensure only opted-in voices get processed. Overlay can be toggled per channel for high-visibility events.
