Build journey

  • Problem

    Cross-language voice chats lose momentum when translation lags.

  • Constraints

    Low-latency, Discord-native, readable directly in chat.

  • Architecture

    Discord voice → streaming STT (Deepgram) → translation → inline bot messages; optional captions overlay.

  • Reliability

    Reconnection guards, rate-limit backoff, interim/final dedupe.

  • Privacy & UI

    Per-user language prefs; opt-in overlay.

  • Status / Next

    Hardening reconnect paths, tuning translation consistency, room-health console.
